hexo for github

1.下载

  • git X86 X64

    运行Git Bash验证： git –version

  • node X86 X64

    验证： node -v 和 npm -v

2.注册Github

新建仓库：user.github.io

3.安装 hexo

安装： npm install hexo-cli -g

验证： hexo version

4.新建项目


hexo init user.github.io
cd user.github.io
npm install

检验： hexo server 或者 hexo s 然后访问 http://localhost:4000

5.关联Github和Hexo

  • 生成SSH-Key： ssh-keygen -t rsa -C “user@email.com” 输入密码或为空，会生成”C:\users\user.ssh\id_rsa.pub”文件，打开复制全部内容，打开https://github.com/settings/ssh新建new SSH Key粘贴进去就好。

  • 检验： ssh -T git@github.com 选择yes看到欢迎语就好了

  • 配置Git个人信息

    用户名： git config –global user.name “user”

    邮箱： git config –global user.email “email”

    配置Deployment，修改文件夹里的 _config.yml文件最后（注：冒号后面有个空格）


    deploy:
    type: git
    repo: git@github.com:user/user.github.io.git
    branch: master

6.本地文件提交到GitHub


npm install hexo-deployer-git –save //安装部署插件
hexo clean //生成新的 public文件
hexo generate //生成新的 public文件
hexo deploy //部署

7.Hexo基本操作

  • 新建文章: hexo new “new post” 可以到其目录编辑 .md 文件，也可以直接在所在的文件夹创建 .md 文件。

  • 文章抬头：


    title: hexo for github
    date: 2017-10-17 16:29:05
    description: How to make a blog #文章描述
    categories: #文章分类
    - 笔记
    tags: #文章标签
    - Github
    - Hexo
    toc: true # 生成目录
    author: yyh0301
    comments:
    original:
    permalink: #指定链接
